# Dots and Boxes (점과 선)

---

## 문제 설명

두 로봇이 7×7 격자의 점들을 잇는 선을 그으며 영역 차지 대결을 펼칩니다.
사각형 4면을 모두 완성한 플레이어가 그 칸을 차지하고, 더 많은 칸을 차지한 플레이어가 승리합니다.

### 게임 규칙

* **Rule 1:** 플레이어는 자신의 턴에 **인접한 두 점 사이의 빈 선분** 하나를 선택합니다.
* **Rule 2:** 선분을 추가했을 때 사각형 4면이 모두 완성되면, 해당 칸을 **점령**합니다.
  * 한 번에 여러 칸을 점령할 수도 있습니다.
  * 박스를 완성한 플레이어는 **한 번 더** 선분을 놓을 기회를 얻습니다.
* **Rule 3:** 모든 선분이 배치되면 게임이 종료됩니다.
* **Rule 4:** 종료 시 박스를 더 많이 점령한 플레이어가 승리합니다. 같으면 무승부입니다.

> **목표:** 박스를 더 많이 점령하는 AI를 작성하세요.

---

## 보드 구조

- **점(dot)**: 7×7 = 49개
- **선분**: 수평 84개 + 수직 84개 = 총 168개
- **박스**: 6×6 = 36개

좌표 기준: 점의 행은 0~6, 열은 0~6

---

## 입/출력 형식

| 커맨드 | 입력 형식 | 설명 |
| :--- | :--- | :--- |
| **READY** | `READY (FIRST\|SECOND)` | 선공(FIRST) 또는 후공(SECOND) 정보. `OK`를 3초 이내에 **출력해야 합니다.** |
| **INIT** | `INIT 6 6` | 보드 크기 정보 (행 수, 열 수). (출력 불필요) |
| **TIME** | `TIME t1 t2` | 내 남은 시간(t1)과 상대 남은 시간(t2). 선분의 두 점 좌표를 `r1 c1 r2 c2` 형식으로 **출력해야 합니다.**<br>- 수평 선분: `r1 c1 r1 c1+1` (r1∈[0,6], c1∈[0,5])<br>- 수직 선분: `r1 c1 r1+1 c1` (r1∈[0,5], c1∈[0,6])<br>- 초기 시간: 10,000ms |
| **OPP** | `OPP r1 c1 r2 c2 t` | 상대가 놓은 선분 좌표와 사용 시간(t). (출력 불필요) |
| **FINISH** | `FINISH` | 게임 종료 신호. 정상 종료해야 합니다. (출력 불필요) |

> :warning: **주의:** 모든 출력 후에는 반드시 버퍼를 flush해야 합니다 (`endl` 또는 `flush`).

---

## 좌표 규칙

- `r1 c1 r2 c2` 형식에서 두 점은 **인접**해야 합니다 (거리 = 1)
- 수평 선분: `r1 == r2`, `c2 == c1 + 1`
- 수직 선분: `c1 == c2`, `r2 == r1 + 1`
- 이미 놓인 선분을 다시 선택할 수 없습니다
