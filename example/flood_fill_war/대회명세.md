# 영역 전쟁 (Flood Fill War)

---

## 문제 설명

아주대학교 캠퍼스 지도 위에서, 두 팀의 치토가 색을 뿌려 영역 확장 대결을 펼칩니다.
지도는 10행 10열의 격자로 나뉘어 있으며, 각 칸에는 **1번부터 6번 사이의 색상**이 칠해져 있습니다.
선공 팀은 좌상단(0, 0)에서, 후공 팀은 우하단(9, 9)에서 출발하여 번갈아 가며 색상을 선택하고 영역을 넓혀 나갑니다.
게임이 끝났을 때 더 많은 칸을 차지한 팀이 승리합니다.

### 게임 규칙

* **Rule 1:** 플레이어는 자신의 턴에 색상 번호(1~6) 하나를 선택합니다.
* **Rule 2:** 내 영역 전체가 선택한 색상으로 변경됩니다.
* **Rule 3:** 내 영역에 상하좌우로 인접한 중립 칸 중 선택한 색상과 같은 칸을 모두 흡수합니다(BFS). 흡수된 칸은 내 영역이 됩니다.
* **Rule 4:** 보드 전체에 중립 칸이 하나도 남지 않으면 게임이 종료됩니다.
* **Rule 5:** 게임 종료 시점에 더 많은 칸을 차지한 플레이어가 승리합니다. 칸 수가 같으면 무승부입니다.

> **목표:** 선공 또는 후공이 되어 더 많은 칸을 차지할 수 있는 프로그램을 작성하세요.

---

## 입/출력 형식

참가자의 프로그램은 한 줄씩 입력을 받아 각 명령에 따라 동작해야 합니다. 출력이 필요한 명령은 아래 표에서 강조 표시되어 있습니다.

| 커맨드 | 입력 형식 | 설명 |
| :--- | :--- | :--- |
| **READY** | `READY (FIRST\|SECOND)` | 선공/후공 정보를 알려주는 명령입니다. 준비 완료를 의미하는 `OK`를 3초 이내에 **출력해야 합니다.** |
| **INIT** | `INIT r0 r1 r2 r3 r4 r5 r6 r7 r8 r9` | 보드 초기 상태를 알려주는 명령입니다. 10개의 문자열이 주어지며, 각 문자열은 10자리 숫자(1~6)로 구성됩니다. `r0`은 0번 행, `r9`는 9번 행에 해당합니다. (출력 불필요) |
| **TIME** | `TIME t1 t2` | 각 플레이어의 남은 시간을 알려주는 명령입니다. `t1`은 내 남은 시간(ms), `t2`는 상대 남은 시간(ms)입니다.<br>이 명령을 받으면 선택할 색상 번호를 `c` 형식으로 **출력해야 합니다.**<br>- `c`는 1 이상 6 이하의 정수입니다.<br>- 처음 주어지는 시간은 10,000(ms)입니다. |
| **OPP** | `OPP c t` | 상대방이 선택한 색상과 해당 턴에 사용한 시간 `t`(ms)를 알려주는 명령입니다. (출력 불필요) |
| **FINISH** | `FINISH` | 게임 종료를 알리는 명령입니다. 이 명령을 받으면 프로그램이 정상 종료되어야 합니다. (출력 불필요) |

> :warning: **주의사항:** 모든 출력 이후에는 반드시 버퍼를 flush해야 합니다. 버퍼를 flush하지 않으면 출력이 전달되지 않아 TLE(시간 초과)가 발생할 수 있습니다.

---

## 보드 상세 설명

* 보드는 10행 × 10열 격자입니다.
* 각 칸은 **색상(1~6)** 과 **소유자(0=중립, 1=선공, 2=후공)** 정보를 가집니다.
* 게임 시작 시 `(0, 0)` 칸은 선공 소유, `(9, 9)` 칸은 후공 소유이며, 나머지는 모두 중립입니다.
* 두 시작 코너의 초기 색상은 서로 다릅니다.

---

## 로그 형식

채점기(`judge`)는 게임 진행 중 표준 출력으로 다음 형식의 로그를 출력합니다.

```
INIT r0 r1 r2 r3 r4 r5 r6 r7 r8 r9
FIRST 3
SECOND 5
FIRST 1
SECOND 2
...
WIN LOSE
```

* `INIT` 줄: 각 ri는 10자리 색상 문자열(1~6).
* `FIRST c` / `SECOND c`: 해당 플레이어가 색상 `c`를 선택한 턴.
* 마지막 줄: 두 플레이어의 결과 (`WIN`, `LOSE`, `NONE`, `TIME_LIMIT`, `ERROR`).

## 컴파일 및 실행 예시

```bash
# 컴파일
g++ -O2 -o judge judge.cpp
g++ -O2 -o sample sample_code.cpp

# 실행
./judge "./sample" "./sample"
```

## 예시 입출력 (sample_code 기준)

### 선공으로 시작할 때

```
입력                          출력
READY FIRST               →  OK
INIT 1234561234 ...       →  (없음)
TIME 10000 10000          →  3
OPP 5 12                  →  (없음)
TIME 9988 9985            →  2
...
FINISH                    →  (없음)
```

### 후공으로 시작할 때

```
입력                          출력
READY SECOND              →  OK
INIT 3456123456 ...       →  (없음)
OPP 3 10                  →  (없음)
TIME 10000 9990           →  6
...
FINISH                    →  (없음)
```
